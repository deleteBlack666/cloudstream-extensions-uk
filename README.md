<!-- Brief information about the extension -->
## 📖 Що це таке?
Це спеціальне розширення для перегляду фільмів, серіалів та аніме в якісному українському дубляжу від різних постачальників в стрімінговій програмі [Cloudstream](https://github.com/recloudstream/cloudstream).

<!-- Installation guide -->
## ⚙️ Інсталяція
Скопіюйте посилання нижче і перейдіть в Додаток -> Параметри -> Розширення -> Додати репозиторій

https://raw.githubusercontent.com/deleteBlack666/cloudstream-extensions-uk/master/repo.json

Відомі проблеми:
- Значно довше завантаження епізодів на тайтлах 300+ серій через HTTPS-запити для отримання прев'ю через Ashdi
- Значно довше завантаження епізодів на тайтлах 1000 серій через запити для отримання прев'ю через Moon
- При 600+ серіях може дуже довго починатися відтворення епізоду (особливо помітно на плеєрі Moon)
- На великих тайтлах (наприклад One Piece) при першому завантаженні може виникати таймаут, але після натискання «Повторити з'єднання» епізоди завантажуються нормально
